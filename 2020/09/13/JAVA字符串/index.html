    字符串常用方法

    查找字符indexOf方法

    indexOf(“字符串”)，其可以起到过滤信息的作用。其接受一个字符串，然后将一段文本中第一个的匹配到的坐标索引值返回。得到一个int类型的返回值，-1为未找到匹配字符串。

    若要寻找第二个出现的相应字符串的位置，则使用indexOf第二种调用方式，indexOf(“字符串”, “开始索引坐标值”)

    字符串拼接substring

    substring可以截取设置的位置之后的内容，两种调用方式。第一种，substring(“开始索引坐标”, “结束索引坐标”)，其包含开始索引的值，但不包含结束索引的值。第二种，substring(“开始索引坐标”).

    字符串开始和结束内容判断startsWith/endsWith

    endsWith(“字符串”)，其接受一个字符串，判断文本是否已该字符串结束。startsWith类似。

    字符串替换replaceAll

    replaceAll(“要替换的值”, “新值”)，其替换文本中所有出现的要替换的值，旧文本不变。

    其还有种用途为删除字符串

    字符串分割split

    split(“分割字符串”)，根据给与的特殊字符来分割字符串

    大小写转换toUpperCase/toLowerCase

    toUpperCase()，将字符串中的字母均转换为大写


    toLowerCase类似

    字符串比较equals

    字符串比较不是使用==，而是使用equals(“被比较字符串”)方法

    数字和字符串转换Integer.parseInt

    数字转换为字符串两种方法，第一种为 +


    第二种为String.valueOf()方法，其接受数字，浮点，布尔类型转换为字符串
