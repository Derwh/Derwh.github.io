    rgb形式

    原理与十六进制相同，颜色由red，green，blue三色深浅组合而成，范围为0~255，数字越大越深。

    rgba形式

    rgba相比较rgb多了a，其为透明度，范围为0~1，数字越小透明度越高。小于1的数字0.x可简写为.x。

    文字对齐方式

    文字行高

    行高作用—改变段落中行与行之间的距离

    如上图，行间距为8px。

    行高作用—使文字上下居中

    随着行高的变化，文字的位置在不断变化，若使行高与外部盒子的高度相同，则可使文字上下居中。

    字间距

    字间距为文字之间的距离，中英文有不同的含义，中文是指汉字之间的距离，英文是指字母之间的距离，不是单词之间的距离。

    字体

    多值的情况下按序匹配，取决于使用者的电脑是否安装该字体，没有则使用默认的微软字体。

    CSS引入方式

    CSS的三种引入方式

    行内样式

    直接在标签内通过style属性引入为行内样式，使标签过于冗长且没有复用的能力，不建议。

    内部样式

    在head标签内添加style标签，将样式加入其中。使头部过于冗长，可能会头重脚轻，不建议。

    外部样式

    新建一个css文件，将样式从html中抽离放在其中。在head标签里添加link标签引入。


    rel属性规定了当前文档和被链接文档之间的关系，其值stylesheet意思是文档的外部样式表，记住即可。
    type属性规定了被链接文档的MIME(多用途互联网邮件拓展类型)类型，其最常见值为text/css，该类型描述样式表。
    href属性的值是要引入的链接地址。

    相对路径

    含义为相对当前文件去寻找目标文件。

    常用选择器

    选择器的层叠性

    以标签选择器为例，相同的标签渲染中相同的属性，后写的会覆盖先写的。类和id选择器均适用。

    标签选择器

    之前使用的形式即为标签选择器形式


    其渲染整个标签，减少了重复代码，但是缺少针对性。

    类选择器

    一个标签可以添加多个类名，定义类中类名顺序不会导致属性覆盖，但是css中的顺序会导致覆盖。

    id选择器

    同一个id在文档中只能出现一次，一个标签只能定义一个id。

    高级选择器

    后代选择器

    侧重后代的概念，一个标签内的所有标签都是其后代


    对于多层嵌套的情况，可以有多种方法，侧重于选中这个概念。

    交集选择器

    上例选中的即为电子产品和家装服务。

    子选择器

    上例使用子选择器，1颜色为orangered，2、3颜色为black。
    相比较后代选择器，子选择器侧重子这个概念。

    并集选择器

    为不同标签添加相同样式。

    单个选择器的优先级

    id选择器>类选择器>标签选择器

    文字属性的继承性

    默认情况下文字颜色为黑色，加了css渲染后


    虽然只是选中了ul进行修饰，但是li中文字也变了颜色，体现了文字属性具有继承性，包括文字大小，文字粗细，字体等。

    高级选择器的优先级
    权重计算法

    设置id选择器的权重为100，类选择器的权重为10，标签选择器的权重为1。通过权重的大小决定选择器的优先级，权重越大，优先级越高。
    权重相同才会考虑层叠性
    考虑权重之前还应考虑是否选中

    CSS盒模型

    盒模型—content

    div标签

    div标签是块状标签，一个干净透彻的盒子，不做修饰的情况之下没有高度，宽度默认和父标签宽度一致。

    width/height

    设置宽高


    百分比尺寸

    其中%是相对于父标签的。
    当需要将盒子铺满整个浏览器的时候可以设置为100%
    默认情况下body高度为0px，宽度和页面一样

    盒模型—padding

    padding即为内边距

    padding分开写

    默认情况下padding为0

    padding简写

    第一个值是上，第二个是右，第三个若没有则和第一个一致，第四个若没有则和第二个一致。

    上下一样，左右一样
    上下一样，左右不一样
    上下不一样，左右一样
    box-sizing

    box-sizing规定了如何计算一个元素的总宽度和高度。有两个值，content-box和border-box，默认是content-box。

    content-box

    content-box尺寸计算公式：
    width = 内容宽度
    height = 内容高度

    border-box

    border-box尺寸计算公式
    width = border + padding + 内容宽度
    height = border + padding + 内容高度

    盒模型—border

    设置框线
    圆角
    阴影

    阴影实现的原理可以看成在矩形下有一个相同的重叠的矩形，当其偏移时就会产生阴影效果。
    x偏移量表示下矩形左右偏移大小，正数为右。
    y偏移量表示下矩形上下偏移大小，正数为下。
    阴影模糊半径表示边线的清晰度。
    阴影扩散半径表示向外延展。
    阴影颜色表示下矩形背景色。

    盒模型—margin

    设置margin

    和padding类似

    两个盒子之间margin的计算

    水平距离为两个盒子左右margin之和
    垂直距离为两个盒子上下margin的最大值

    盒子左右居中

    margin可以使子盒在父盒中左右居中，但是前提是子盒须有width属性。

    盒模型—display: block/none

    块状元素和行内元素

    块状元素独占一行，可以设置宽高，而行内不可以设置宽高。

    行内转块状
    块状转行内
    标签消失

    盒模型—display: inline/inline-block

    inline

    行内元素不能设置宽高，可以设置padding(不能用这种方法来设置块的大小)，可以设置左右margin，不能设置上下margin。

    inline-block

    其就是设置一个同时具有块状和行内元素性质的元素，不会独占一行的块状元素。

    空白折叠现象

    将div的display设置为inline-block时，会出现两个盒子之间有空白，但是却没有设置margin，这就是空白折叠。
    出现现象的原因是两个div中存在回车，在html中将回车视为一个字母，所以会出现空白现象。

    去除方法1

    去除回车

    去除方法2

    给父元素添加word-spacing属性，其意思为单词之间的距离，将其写为负数即可，一般设置为小于-20px。

    去除方法3

    回车可以看成一个文字，给父元素设置font-size: 0px，空白即会消失。

    CSS定位

    position这个属性在CSS中用于定位DOM元素，修改DOM元素的布局

    Position-static

    默认定位，浏览器会给所有没有设置position属性的DOM元素添加position=static。当设置top，left，bottom，right会导致文档流变化。

    Position-relative

    相对定位


    其left等调整都是相对于当前元素static布局位置进行的调整，不改变页面布局。

    Position-absolute

    绝对定位
    设置之后其会脱离当前文档流，形成新的图层。绝对定位不为元素预留空间，通过指定元素相对于最近的非static定位的祖先元素的偏移来确定元素位置。
    可通过将父元素设置position=relative来控制。

    Position-fixed

    固定定位
    固定定位不为元素预留空间，而是通过设置元素相对于屏幕视口(viewport)来指定元素的位置，元素的位置在屏幕滚动的时候不会改变。

    z-index

    通过设置z-index可以改变图层优先级。
    所有非static元素的z-index均为0，z-index越大，图层越靠近观察者，同样的z-index，越在后面越靠近观察者。

    Position-sticky

    粘性定位
    当页面滚动时，非其所在页面会固定在顶部，在其所在页面会回到自己的位置。
