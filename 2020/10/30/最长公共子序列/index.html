最长公共子序列
发表于 2020-10-30| 更新于 2020-10-30|算法--动态规划
字数总计:772|阅读时长: 3 分钟

问题描述

一个字符串A的子串被定义成从A中顺次选出若干个字符构成的串。如A=“cdaad” ,顺次选1，3，5个字符就构成子串” cad” ,现给定两个字符串，求它们的最长共公子串

输入

第一行两个字符串用空格分开。两个串的长度均小于2000 。

输出

最长子串的长度

样例输入

样例输出

分析

为了得到公共字符个数，需要同时遍历两个字符串，将每一个字符进行比较，判断是否相同。而为了得到总的相同个数，需要将之前的个数记录下来。如果当前位置的两个字符相同，那么此时公共字符个数就是将这两个字符均在其对于字符串去掉之后的公共字符长度+1。即为：c[i][j] = c[i - 1][j - 1] + 1。如果不相等，就将两个字符分别在对于字符串中去掉，检查去掉哪个之后公共字符最多，得到的结果取最大的那个。

理论实现

对于样例
定义c数组保存公共字符情况，行为第二个字符串，列为第一个字符串

0 1 2 3 4
0 0 0 0 0 0
1 0 1 1 1 1
2 0 1 1 1 1
3 0 1 1 2 2
4 0 1 1 2 2
5 0 1 1 2 3

可以得到最长为3

易错点

注意数组的长度问题，因为标号方便的原因，存储字符的数组是从1开始存储的，后面取用的时候需注意。

拓展

题目中没有要求输出最长公共子序列，若要输出，则还需定义一个数组b，记录当前c中数字来源情况，比如将两个字符相同的情况记为b[i][j] = 1。后面可以从b中最后一个数字往前推，推导过程中标号为1的地方即为公共字符的地方。
